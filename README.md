# 项目简介

基于深度学习YOLOv12的设备泄漏检测系统，运用先进的目标检测技术，实现对工业设备泄漏的有效监测。系统包含功能模块：数据预处理、YOLOv12检测模型、结果展示UI界面以及登录注册安全机制。技术特点在于采用YOLO数据集优化模型准确性，支持实时检测与报警，提高工业生产安全水平。项目源码为开发与二次研究提供便利。


## 核心功能
- 数据预处理：对工业设备图像进行预处理，优化输入质量
- YOLOv12检测模型：基于深度学习的目标检测技术，实现设备泄漏识别
- 结果展示UI界面：直观展示检测结果和设备状态
- 登录注册安全机制：确保系统访问安全
- 实时检测与报警：及时发现设备泄漏问题
- 支持YOLO数据集：使用专门数据集优化模型准确性

## 适用场景
- 工业生产环境中的设备泄漏监测
- 工业安全管理系统
- 设备状态实时监控平台
- 需要自动化检测工业异常场景的应用

## 技术特点
- 基于YOLOv12深度学习框架构建
- 采用专门优化的YOLO数据集
- 支持实时处理与响应
- 提供完整项目源码，便于二次开发与研究
- 具备完整的前端UI界面，提升用户体验
